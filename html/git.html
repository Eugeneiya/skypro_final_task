Git

  • git status в терминале VS Code - просмотр изменений в test-repo.
  • git add index.html - добавление файла index.html в файлы, отслеживаемые Git.
  • git commit -m "first commit" - выполнение коммита для сохранения изменений в файле в репозитории.
  • git push - внесение изменений на удаленный репозиторий.
  • git log - выведение списка всех коммитов/
